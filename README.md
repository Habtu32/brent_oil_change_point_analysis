# Brent Oil Change Point Analysis

This repository contains materials for a change point analysis of historical Brent crude oil prices, with a focus on linking structural breaks to geopolitical and economic events.

## Task 1 Deliverables
- Analysis workflow and assumptions: `docs/task1_analysis_plan.md`
- Key geopolitical and policy events: `data/key_events.csv`

## Repository Layout
- `data/`: event data and supporting datasets
- `docs/`: written plans and reports
- `src/`: data loading and cleaning utilities
